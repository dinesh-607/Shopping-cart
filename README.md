NeroFlix – OTT Streaming Platform (Frontend Project)

NeroFlix is a modern OTT (Over-The-Top) streaming platform designed and developed using HTML, CSS, and JavaScript. This project focuses on delivering a visually engaging and user-friendly interface inspired by leading streaming services.

The platform features a sleek dark-themed UI with dynamic layouts, allowing users to explore movies, series, and upcoming content seamlessly. Key functionalities include interactive navigation menus, responsive design for multiple devices, and engaging call-to-action sections such as “Start Watching” and user authentication buttons.

Through this project, I strengthened my skills in frontend development, including responsive web design, UI/UX principles, and DOM manipulation using JavaScript. NeroFlix demonstrates my ability to build real-world web interfaces with clean design, smooth user experience, and scalable structure.

Key Features:

Responsive and modern OTT-style user interface
Dynamic navigation and interactive elements
Organized content sections (Movies, Upcoming, Languages)
Clean and reusable code structure
Optimized for performance and usability

Technologies Used: HTML5, CSS3, JavaScript
